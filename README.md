# 项目简介

## 核心功能
- 基于YOLOv8的防护工具及破损检测系统
- 支持10类目标检测：破损安全手套、破损安全帽、破损安全鞋、无安全手套、无安全帽、无安全鞋、人物、安全手套、安全帽、安全鞋
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整的标注数据集（13000张）
- 包含训练好的模型及训练结果
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供界面功能：实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 提供完整的UI界面源文件

## 适用场景
- 工业安全防护设备检测
- 工作场所安全合规检查
- 安全装备质量监控
- 自动化安全检测系统

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖技术栈：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9等
- 采用YOLOv8深度学习模型
- 提供完整的运行教程和项目文件
- 配置成功环境可直接使用
